## Downloads

- **Linux (x86_64)** — `NAMix-__VERSION__-linux-x86_64.tar.gz`, containing both the VST3 plug-in and the standalone app
- **Demo model** — `muffinator_v4_optimal.param.nam`, for trying the parametric knobs (no model is bundled)

## Before you install

Requires **glibc 2.35 or later** (Ubuntu 22.04+, Debian 12+, Fedora 36+, and most current rolling-release distros). Ubuntu 20.04, Debian 11, RHEL/CentOS 9, and openSUSE Leap 15.x ship an older glibc and can't load these binaries -- build from source instead (see the README).

Extract the tarball, then either copy the `.vst3` folder into `~/.vst3/` and rescan plug-ins in your DAW, or run the standalone binary directly.
